FAQ

재생목록의 최대 길이가 있습니까?

엄격한 제한은 없지만 매우 큰 재생목록(500개 이상의 동영상)은 완전히 처리하는 데 몇 분이 걸릴 수 있습니다.

재생목록이 순서대로 다운로드됩니까?

예. 원래 재생목록 순서는 기본적으로 유지됩니다.

재생목록의 동영상 수에 제한이 있습니까?

이론적으로 모든 크기의 재생목록을 구문 분석할 수 있지만 최적의 성능과 브라우저 안정성을 위해 한 번에 200개 이하의 동영상이 있는 재생목록을 다운로드하는 것이 좋습니다.

하나의 거대한 파일로 다운로드됩니까 아니면 별도의 파일로 다운로드됩니까?

재생목록의 각 동영상은 자체 별도 파일로 변환 및 다운로드됩니다. 단일 대규모 오디오/비디오 파일로 병합하지 않습니다.

'나중에 볼 동영상' 재생목록을 다운로드할 수 있습니까?

'나중에 볼 동영상' 재생목록은 계정에 비공개입니다. 서버에서 액세스할 수 없습니다. 먼저 해당 동영상을 공개 또는 목록에 없음 재생목록으로 이동해야 합니다.

재생목록에 라이브 스트림이 포함되어 있으면 어떻게 됩니까?

라이브 스트림이 종료되고 VOD로 제공되는 경우 정상적으로 다운로드됩니다. 스트림이 현재 라이브 중인 경우 유한한 길이가 없으므로 도구가 건너뜁니다.

50개 동영상 재생목록을 변환하는 데 얼마나 걸립니까?

목록을 구문 분석하는 데는 몇 초밖에 걸리지 않습니다. 변환 시간은 동영상의 총 길이와 선택한 품질에 따라 다르지만 동시 처리는 일반적으로 일반적인 음악 재생목록을 3분 이내에 완료합니다.

다운로드를 50번 클릭해야 합니까?

현재 브라우저 보안 제한으로 인해 웹사이트가 한 번에 50개의 파일 다운로드를 자동으로 트리거할 수 없습니다. 생성된 각 파일에 대해 다운로드 버튼을 클릭해야 합니다.

재생목록 썸네일을 다운로드합니까?

개별 동영상 파일에는 각각의 동영상 썸네일이 메타데이터(오디오 형식의 경우) 또는 포스터 프레임(비디오 형식의 경우)으로 포함됩니다.

채널의 전체 동영상 목록을 다운로드할 수 있습니까?

채널에 '모두 재생' 버튼이 있거나 모든 업로드가 포함된 재생목록이 있는 경우 가능합니다. 해당 특정 재생목록 URL을 복사하기만 하면 됩니다.

파일에 번호가 매겨져 있습니까?

파일은 원래 YouTube 제목을 유지합니다. 제작자가 제목에 번호를 매긴 경우(예: 'Episode 1: ...') 해당 번호가 다운로드한 파일에 포함됩니다.

동영상의 절반만 나타난 이유는 무엇입니까?

이는 일반적으로 누락된 동영상이 비공개, 삭제 또는 지역 차단되었음을 의미합니다. 우리 도구는 공개적으로 액세스 가능한 미디어만 가져올 수 있습니다.
