FAQ

May maximum playlist length ba?

Walang hard cap, ngunit ang napakalalaking playlists (500+ videos) ay maaaring tumagal ng ilang minuto upang maproseso nang buo.

Ang mga playlist ba ay dina-download nang may pagkakasunod-sunod?

Oo. Ang orihinal na pagkakasunod-sunod ng playlist ay napapanatili by default.

May limitasyon ba sa bilang ng mga video sa playlist?

Maaari kaming teoretikal na mag-parse ng playlists ng anumang laki, ngunit para sa optimal performance at browser stability, inirerekomenda namin ang pag-download ng playlists na may 200 videos o mas kaunti sa isang pagkakataon.

Ito ba ay mag-download bilang isang malaking file o hiwalay na files?

Bawat video sa playlist ay kino-convert at dina-download bilang sariling hiwalay na file. Hindi namin pinagsasama ang mga ito sa isang malaking audio/video file.

Maaari ko bang i-download ang 'Watch Later' playlist?

Ang iyong 'Watch Later' playlist ay pribado sa iyong account. Ang aming mga server ay hindi maaaring ma-access ito. Kailangan mong ilipat muna ang mga video na iyon sa isang Public o Unlisted playlist.

Ano ang mangyayari kung ang playlist ay naglalaman ng live stream?

Kung ang live stream ay natapos na at available bilang VOD, ito ay normal na mag-download. Kung ang stream ay kasalukuyang live, ang tool ay lalaktaw dito, dahil wala itong finite length.

Gaano katagal ang 50-video playlist upang ma-convert?

Ang pag-parse ng listahan ay tumatagal lamang ng ilang segundo. Ang conversion time ay depende sa kabuuang haba ng mga video at sa napiling kalidad, ngunit ang aming concurrent processing ay karaniwang nagtatapos ng isang typical music playlist sa loob ng 3 minuto.

Kailangan ko bang mag-click ng download 50 beses?

Sa kasalukuyan, ang mga browser security restrictions ay pumipigil sa mga website na awtomatikong mag-trigger ng 50 file downloads nang sabay-sabay. Kailangan mong i-click ang download button para sa bawat generated file.

Dina-download ba nito ang playlist thumbnail?

Ang indibidwal na video files ay magkakaroon ng kani-kanilang video thumbnails na naka-embed bilang metadata (para sa audio formats) o bilang poster frame (para sa video formats).

Maaari ko bang i-download ang buong video list ng channel?

Kung ang channel ay may 'Play All' button o isang playlist na naglalaman ng lahat ng kanilang uploads, oo. I-copy lang ang specific playlist URL na iyon.

Ang mga file ba ay may numero?

Ang mga file ay pinapanatili ang kanilang orihinal na YouTube titles. Kung ang creator ay nag-numero ng mga pamagat (hal., 'Episode 1: ...'), ang numerong iyon ay magiging nasa iyong na-download na file.

Bakit kalahati lang ng mga video ang lumabas?

Ito ay karaniwang nangangahulugan na ang mga nawawalang video ay Private, Deleted, o Region-Blocked. Ang aming tool ay maaari lamang kumuha ng publicly accessible media.
